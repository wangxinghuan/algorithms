##find_criminal.c

协助破案。

假设已经查清，有A,B,C,D,E五个疑犯可能参与制造了一起银行的抢劫案，但是不知道其中到底哪几个是真正的案犯。不过，有
以下数据：

1. 如果A参与了作案，则B一定参与。
2. B和C两人中只有一人参与了作案。
3. C和D要么都参与了作案，要么都没有参与
4. D和E两个人中至少有一人参与作案。
5. 如果E作案，则A和D一定参与作案。
